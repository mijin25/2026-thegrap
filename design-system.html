Language Font Rules

`html:lang(ko)`
PPNeueMontreal 우선 + 한글은 Pretendard로 자연 fallback
ko 자간/행간 보정
`p/li`는 `letter-spacing: -0.01em`, `line-height: 1.6`
`span`은 `letter-spacing: -0.01em` (단, `[class*="logo"]` 제외)
`span:not([class*="logo"])` 규칙이 곧 “기본 한글 타이포 감”을 만듦
`html:lang(en)`
PPNeueMontreal 우선 (영문 기본)
`html:lang(zh-CN)`
PPNeueMontreal + Noto Sans TC + PingFang SC
`html:lang(ja)`
PPNeueMontreal + Noto Sans JP + Hiragino Sans

Layout Tokens

`--gutter`
`clamp(16px, 2.5vw, 36px)` — 페이지 좌우 여백 기준
Subpage Hero
`.page-hero` + `.page-title--copy` + `.hero-sub-text`를 공통 규칙으로 사용

Header Rules

`#header`
`position: fixed` / `z-index: 900` / `display: grid (1fr auto 1fr)`
`pointer-events: none` + 자식은 `pointer-events: all`
`mix-blend-mode: difference`로 발광 레이어 위 가독성 확보
`#desktop-nav a`
`font-family: PPNeueMontreal` / `text-transform: uppercase`
기본 색: `var(--c-on-bg)`, hover: `var(--c-primary)`
모바일 분기
`@media (max-width: 768px)`에서 `#desktop-nav` 숨김 + `#hamburger` 표시
`lang-switcher` 숨김

Subpage Hero Rules

`.page-hero`
`padding: clamp(140px,18vw,220px) var(--gutter) clamp(60px,8vw,100px)`
`.page-title`
`font-size: clamp(4rem,11vw,10rem)` / `text-transform: uppercase`
`letter-spacing: -.02em !important`
`.page-title span`은 기본적으로 `translateY(110%)`
`.hero-sub-text`
`font-size: clamp(0.82rem, 1.1vw, 1rem)` / `line-height: 1.65`
데스크톱: `max-width: min(64ch, 100%)`
ko: `PPNeueMontreal 우선 + Pretendard fallback`
`.reveal`
`opacity: 0; transform: translateY(20px);` (스크롤 진입 시 gsap에서 노출)
